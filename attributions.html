Attributions

Every Cinema London is a showcase built on data and services generously provided by others. Thank you to the following.

Performance data — Clusterflick

Cinema listings and showtimes (performances, venues, formats and accessibility details) are compiled by Clusterflick, which aggregates what’s on across London’s cinemas and venues.

This project uses the published combined dataset from github.com/clusterflick/data-combined.

Film metadata — The Movie Database (TMDB)

Film metadata — titles, posters, genres, classifications, runtimes and directors — is sourced from The Movie Database (TMDB).

This product uses the TMDB API but is not endorsed or certified by TMDB.

Grid — AG Grid

The interactive data grid is AG Grid (Enterprise), used here to demonstrate row grouping, pivoting and filtering over the dataset.

Source code: github.com/alistairjcbrown/everycinema.london
